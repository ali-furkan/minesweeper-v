## Installation

You can download it from the [release page](./releases) or run it on your pc with `v -autofree run .`

## Controllers

|Control        |Description        |
|---------------|-------------------|
|Q-key          |Change click mode  |
|R-key          |Restart the game   |
|Mouse Button 1 |Click the cell     |
